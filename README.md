# Rock Paper Scissors Game

A simple and interactive Rock Paper Scissors game built with HTML, CSS, and JavaScript. The game allows users to play against the computer with randomly generated choices and displays the result instantly.

## Features

-  Play against the computer
-  Rock,  Paper,  Scissors options
-  Random computer selection
-  Instant win, lose, or draw result
-  Responsive and user-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/fa-ri-a/JsGame.git
   ```

2. Open the project folder.

3. Open `index.html` in your browser.
